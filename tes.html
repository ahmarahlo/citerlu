🚨 WAR ROOM: STRATEGI & URUTAN SAAT UJIAN (BACA INI DULU!)

Penguji sering kali melihat PROSES kerjamu. Kalau kamu asal ketik tanpa urutan yang jelas, begitu ada error kamu akan stuck dan stress. Gunakan urutan (SOP) ini saat ujian:

TAHAP 1: URUTAN PENGERJAAN SAAT UJIAN (SANGAT PENTING)

  1. (15 Menit) Setup Awal: Buka XAMPP/MySQL. Buat database.sql, import, lalu ketik koneksi.php. Tes buka koneksi.php di browser, kalau layar browser blank / background warna hitam berarti KONEKSI SUKSES. Jangan lanjut sebelum koneksi aman.
  2. (30 Menit) Security / Autentikasi: Kerjakan login.php, cek_login.php, logout.php, dan sebuah dashboard.php kosong yang cuma ada tulisan "Selamat Datang". Pastikan proses Login dan Logout jalan sempurna. Jika ini jalan, asesor sudah lihat sistemmu punya pondasi kuat.
  3. (20 Menit) TRIK DUPLIKASI (COPY-PASTE): Kerjakan form_hewan.php sampai selesai dan rapi (HTML & Tabelnya). Setelah selesai, JANGAN MENGETIK ULANG UNTUK FORM LAIN! Cukup Copy file form_hewan.php tersebut, lalu Rename (Ubah Nama) menjadi form_booking.php dan form_stok.php. Kamu hanya tinggal mengubah text input name dan kolom tabelnya saja. Ini menghemat hafalan HTML-mu hingga 70%!
  4. (60 Menit) Jantung CRUD: Kerjakan aksi.php. Ini file tunggal penentu. Fokus bikin logika INSERT yang gampang dulu (Tambah hewan & stok). Jika sudah masuk ke DB, baru mikirin logika susah (Kuota 15 antrean & Potong Stok Obat).
  5. (45 Menit) Final Boss (Dashboard): Kembali ke dashboard.php. Lakukan query JOIN tabelnya, dan masukkan perhitungan matematika (Diskon 15% + PPN 11%) di dalam while.

TAHAP 2: HAFALAN APA YANG BISA "DIKARANG"?

  • CSS: Jangan dihafal! Buang waktu. Penguji tidak menilai web design, mereka menilai FUNGSI. Gunakan tag tabel bawaan browser, atau kasih border="1" di HTML. Untuk style, karang seadanya asal gak acak-acakan.
  • Penamaan Variabel: Jangan capek-capek pusing hapalin variabel $_POST['tgl_pesan']. SAMAKAN SAJA tulisan di atribut HTML name="tanggal" dengan nama kolom di Database MySQL (tanggal) dan nama receiver di PHP $_POST['tanggal']. Sinkronkan semua 1 nama, hapalanmu selesai!

TAHAP 3: PERTOLONGAN PERTAMA KALO PANIK (ERROR)

  • Error: Undefined Index / Undefined Array Key 👉 JANGAN PANIK! Ini 100% dipastikan karena typo. Huruf besar/kecil di form HTML (name="Nama") beda dengan di penangkap PHP ($_POST['nama']). Samakan hurufnya.
  • Error: Call to undefined function mysqli_... 👉 Kalo kejadian begini di awal, Apache/MySQL mu lupa di Start, atau kamu lupa menaruh include 'koneksi.php'; di paling atas struktur file PHP.
  • Klik Submit, tapi Data Tidak Masuk ke Database (Tidak Muncul Error) 👉 Buka tab baru, cek phpMyAdmin. Kalau tetep ga ada, artinya query INSERT mu typo tanda kutip satu ('). Pastikan formatnya: VALUES ('$_POST[nama]', '$_POST[jenis]').

Langkah 1: Siapkan Database database.sql

💡 HAFALAN TAHAP 1: Ingat saja entitasnya: users, hewan, layanan, stok, dan booking. Sisanya logika foreign key biasa.
CREATE DATABASE db_meowgug;
USE db_meowgug;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(100) NOT NULL,
    password VARCHAR(255) NOT NULL,
    role ENUM('Admin', 'Dokter', 'Pelanggan') NOT NULL,
    is_member BOOLEAN DEFAULT 0
);

CREATE TABLE hewan (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama_hewan VARCHAR(100) NOT NULL,
    jenis VARCHAR(100) NOT NULL,
    pemilik_id INT NOT NULL,
    FOREIGN KEY (pemilik_id) REFERENCES users(id) ON DELETE CASCADE
);

CREATE TABLE layanan (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama_layanan VARCHAR(100) NOT NULL,
    harga INT NOT NULL
);

CREATE TABLE stok (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama_obat VARCHAR(100) NOT NULL,
    jumlah INT NOT NULL,
    harga INT NOT NULL
);

CREATE TABLE booking (
    id INT AUTO_INCREMENT PRIMARY KEY,
    tanggal DATE NOT NULL,
    no_antrean INT NOT NULL,
    hewan_id INT NOT NULL,
    layanan_id INT NOT NULL,
    stok_id INT DEFAULT NULL,
    qty_obat INT DEFAULT 0,
    status ENUM('Menunggu', 'Selesai') DEFAULT 'Menunggu',
    FOREIGN KEY (hewan_id) REFERENCES hewan(id),
    FOREIGN KEY (layanan_id) REFERENCES layanan(id),
    FOREIGN KEY (stok_id) REFERENCES stok(id)
);

Langkah 8: Final Boss (Dashboard & Tagihan) dashboard.php

💡 HAFALAN TAHAP 5 (Paling Susah): Lihat baik-baik komentar // RUMUS PERHITUNGAN! Perhitungan Matematika Diskon dan PPN wajib di dalam blok while. Saat ujian, pastikan rumus diskon itu hanya memotong harga Jasa pelayanannya saja, bukan harga Obat.
<?php
session_start();
include 'koneksi.php';
if(!isset($_SESSION['id'])) header("Location: login.php");

$role = $_SESSION['role'];
$user_id = $_SESSION['id'];
?>
<!DOCTYPE html>
<html>
<head>
    <title>Dashboard</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="nav">
        <a href="dashboard.php">Dashboard</a>
        <?php if($role == 'Pelanggan') { ?>
            <a href="form_hewan.php">Data Hewan</a>
            <a href="form_booking.php">Booking Layanan</a>
        <?php } ?>
        <?php if($role == 'Admin') { ?>
            <a href="form_stok.php">Kelola Stok</a>
        <?php } ?>
        <a href="logout.php" style="float: right;">Logout</a>
    </div>

    <div class="container">
        <h1>Dashboard - Role: <?= $role ?></h1>
        <h2>Tabel Antrean</h2>
        <table>
            <tr>
                <th>Antrean</th><th>Tgl</th><th>Hewan</th><th>Layanan</th><th>Status</th><th>Total Tagihan (Inc. PPN)</th><th>Aksi</th>
            </tr>
            <?php
            // Query JOIN Utama
            $sql = "SELECT b.*, h.nama_hewan, u.is_member, l.nama_layanan, l.harga as h_layanan, s.harga as h_obat 
                    FROM booking b 
                    JOIN hewan h ON b.hewan_id = h.id 
                    JOIN users u ON h.pemilik_id = u.id
                    JOIN layanan l ON b.layanan_id = l.id 
                    LEFT JOIN stok s ON b.stok_id = s.id 
                    ";
            // Jika Pelanggan, filter by id
            if($role == 'Pelanggan') $sql .= " WHERE h.pemilik_id = '$user_id'";
            $sql .= " ORDER BY b.tanggal DESC, b.no_antrean ASC";

            $query = mysqli_query($koneksi, $sql);
            while($row = mysqli_fetch_assoc($query)){
                
                // ===================================
                // 🛑 RUMUS PERHITUNGAN (WAJIB INGAT)
                // ===================================
                $biaya_layanan = $row['h_layanan'];
                $biaya_obat = ($row['h_obat'] != null) ? ($row['h_obat'] * $row['qty_obat']) : 0;
                
                $diskon = 0;
                // Diskon 15% dari biaya layanan JIKA MEMBER
                if($row['is_member'] == 1) {
                    $diskon = 0.15 * $biaya_layanan; 
                }
                
                $total_sementara = ($biaya_layanan - $diskon) + $biaya_obat;
                $ppn = 0.11 * $total_sementara;
                $total_tagihan = $total_sementara + $ppn;
                // ===================================

                echo "<tr>
                        <td>".$row['no_antrean']."</td>
                        <td>".$row['tanggal']."</td>
                        <td>".$row['nama_hewan']."</td>
                        <td>".$row['nama_layanan']."</td>
                        <td><b>".$row['status']."</b></td>
                        <td>Rp ".number_format($total_tagihan, 0, ',', '.')."</td>
                        <td>";
                        
                // Aksi Khusus Dokter (Tombol Selesaikan & Form Pilih Obat)
                if($role == 'Dokter' && $row['status'] == 'Menunggu') {
                    echo "<form action='aksi.php?proses=update_status' method='POST' style='display:inline;'>
                            <input type='hidden' name='booking_id' value='".$row['id']."'>
                            <select name='stok_id' style='width:auto;'><option value=''>-- Tanpa Obat --</option>";
                            $q_obat = mysqli_query($koneksi, "SELECT * FROM stok WHERE jumlah > 0");
                            while($o = mysqli_fetch_assoc($q_obat)) {
                                echo "<option value='".$o['id']."'>".$o['nama_obat']."</option>";
                            }
                    echo "  </select>
                            <input type='number' name='qty' placeholder='Qty' style='width:60px;' min='1'>
                            <button type='submit' class='btn btn-success'>Selesai</button>
                          </form>";
                } else {
                    echo "-";
                }
                echo "  </td></tr>";
            }
            ?>
        </table>
    </div>
</body>
</html>

Langkah 12: Pusat Jantung Logika aksi.php

💡 HAFALAN TAHAP 4 (FOKUS UTAMA): Kerjakan file ini JIKA SEMUA BENTUK sudah siap. Fokusmu letak pada nomor 3 (Validasi Max 15 perhari) dan nomor 4 (Potong Stok Obat oleh Dokter). Ini yang akan jadi bahan uji coba dosen penilai.
<?php
session_start(); include 'koneksi.php';
$proses = isset($_GET['proses']) ? $_GET['proses'] : '';

// ==========================================
// 1. TAMBAH/HAPUS HEWAN
// ==========================================
if($proses == 'tambah_hewan') {
    mysqli_query($koneksi, "INSERT INTO hewan (nama_hewan, jenis, pemilik_id) VALUES ('$_POST[nama_hewan]', '$_POST[jenis]', '$_SESSION[id]')");
    header("Location: form_hewan.php");
} elseif($proses == 'hapus_hewan') {
    mysqli_query($koneksi, "DELETE FROM hewan WHERE id='$_GET[id]'");
    header("Location: form_hewan.php");

// ==========================================
// 2. TAMBAH/HAPUS STOK
// ==========================================
} elseif($proses == 'tambah_stok') {
    mysqli_query($koneksi, "INSERT INTO stok (nama_obat, jumlah, harga) VALUES ('$_POST[nama_obat]', '$_POST[jumlah]', '$_POST[harga]')");
    header("Location: form_stok.php");
} elseif($proses == 'hapus_stok') {
    mysqli_query($koneksi, "DELETE FROM stok WHERE id='$_GET[id]'");
    header("Location: form_stok.php");

// ==========================================
// 3. TAMBAH BOOKING (VALIDASI MAX 15)
// ==========================================
} elseif($proses == 'tambah_booking') {
    $tgl = $_POST['tanggal'];
    
    // Hitung dulu tiket hari tersebut
    $cek = mysqli_fetch_assoc(mysqli_query($koneksi, "SELECT COUNT(*) as total FROM booking WHERE tanggal='$tgl'"));
    
    if($cek['total'] >= 15) {
        // Stop code nya (die)
        die("<h3>Kuota Hari ini Penuh! (Maks 15)</h3><a href='form_booking.php'>Kembali</a>");
    } else {
        // Buat No urut antrean = jumlah existing + 1
        $no_antrean = $cek['total'] + 1;
        mysqli_query($koneksi, "INSERT INTO booking (tanggal, no_antrean, hewan_id, layanan_id) VALUES ('$tgl', '$no_antrean', '$_POST[hewan_id]', '$_POST[layanan_id]')");
        header("Location: dashboard.php");
    }

// ==========================================
// 4. PEMERIKSAAN DOKTER (UPDATE & POTONG STOK)
// ==========================================
} elseif($proses == 'update_status') {
    $booking_id = $_POST['booking_id'];
    $stok_id = !empty($_POST['stok_id']) ? $_POST['stok_id'] : 'NULL';
    $qty = !empty($_POST['qty']) ? $_POST['qty'] : 0;
    
    // Cek jika dokter resepkan BANYAK obat dari tabel form
    if($stok_id != 'NULL' && $qty > 0) {
        $sisa = mysqli_fetch_assoc(mysqli_query($koneksi, "SELECT jumlah FROM stok WHERE id='$stok_id'"))['jumlah'];
        
        if($sisa < $qty) {
            die("<h3>Stok Tidak Cukup! Sisa: $sisa</h3><a href='dashboard.php'>Kembali</a>");
        }
        
        // 4.a Update Stok: Kurangi
        mysqli_query($koneksi, "UPDATE stok SET jumlah = jumlah - $qty WHERE id='$stok_id'");
        // 4.b Update Booking Selesai
        mysqli_query($koneksi, "UPDATE booking SET status='Selesai', stok_id='$stok_id', qty_obat='$qty' WHERE id='$booking_id'");
    } else {
